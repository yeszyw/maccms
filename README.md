- 下载请先阅读“模板说明”或“help”文件。
- 模板并非免费提供，影视模版(**10RMB或2USDT一套**)，叉站模版(**50RMB或8USDT一套**)。
- 收费是因为我花费了大量时间来收集和整理MacCMS模板，这是不可忽视的成本。如不认可，请离开。
